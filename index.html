Getting Started

Add the OpenClaw Helm repository:

helm repo add openclaw https://caninehq.github.io/openclaw-helm
helm repo update

Deploy OpenClaw:

helm install openclaw openclaw/openclaw \
  --set secret.data.OPENCLAW_GATEWAY_TOKEN=<your-token> \
  --set ingress.enabled=true \
  --set ingress.hosts[0].host=openclaw.example.com
